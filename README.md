## Getting started
1. Install Docker and Docker Composer
2. Install Node.JS and npm
   - Node.JS v14.0.0
   - npm v8.1.2

## Start development service

To start MongoDB cluster:
```
docker-compose up
```

To start API, web application and web:
```
npm run start
```
